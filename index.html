CSS: keyframes animations

CSS keyframes animations allow you to create complex animations by defining keyframes at specific points in time. In this example, we've defined an animation named bounce that moves an element up and down.

@keyframes bounce {
0%, 100% {
  transform: translateY(0);
  }
  50% {
  transform: translateY(-30px);
  }
}
CSS: 3D rotation effect to an
element when hovered

This code demonstrates how to apply a 3D rotation effect to an element when hovered, creating a flip animation. You can use this code to display 3D CSS effects on your web page.

.element {
transform: perspective(1000px)
rotateY(45deg);
transition: transform 0.5s;
}
.element:hover {
transform: perspective(1000px)
rotateY(180deg);
}
CSS: Linear Gradient & Radial Gradient

Linear Gradient: The linear-gradient function creates a gradient that transitions from one color to another in a linear direction.Radial Gradient: The radial-gradient function creates a circular gradient that transitions from one color to another. 

/* Linear Gradient */
.background {
background: linear-gradient
(to right, #FFCC00, #FF3300);
}
/* Radial Gradient */
.background {
background: radial-gradient
(circle, #33CC33, #FF9900);
}

Git: git diff

The git diff command is used to display the differences between two commits or between the working directory and the last commit. In the example, changes between two commits are shown. Lines prefixed with '-' indicate deletions, while lines prefixed with '+' indicate additions. This format provides a clear visual representation of code changes.

git diff
Example:
-Lines prefixed with '-' are deletions.
-Lines prefixed with '+' are additions.
-Changes between the two commits are displayed.
